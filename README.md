# Bang! 온라인

보드게임 **Bang!**을 웹 브라우저에서 온라인 멀티플레이로 즐기는 프로젝트입니다.
모바일·PC 브라우저 모두에 최적화되어 있으며, 방 코드로 친구를 초대해 4~7명이 실시간으로 플레이합니다.

## 기술 스택

- **프론트엔드**: React + TypeScript + Vite (모바일 우선 반응형 UI, 한국어)
- **서버**: Node.js + Socket.IO (권위적 게임 상태, 실시간 동기화)
- **공유 로직**: TypeScript 게임 엔진 (`shared`) — 서버·클라이언트 공용

치팅 방지를 위해 게임 상태는 서버가 단독으로 보유하며, 각 플레이어에게는
자신의 손패만 보이도록 **가려진 뷰(redacted view)** 만 전송합니다.

## 모노레포 구조

```
bang-web/
  shared/   게임 규칙 엔진 (역할/덱/턴/거리/승리 판정) + vitest 테스트
  server/   Socket.IO 서버 (방·로비·재접속, 엔진 연결, 브로드캐스트)
  client/   React 클라이언트 (홈/대기실/게임 테이블)
```

## 실행 방법

### 개발 모드

```bash
npm install
npm run dev
```

- 서버: http://localhost:3001
- 클라이언트(Vite): http://localhost:5173

브라우저 탭을 여러 개 열어(또는 같은 와이파이의 휴대폰에서 PC IP로 접속) 한 명이
**방 만들기**, 나머지는 **코드로 입장** 후 방장이 **게임 시작** 을 누르면 됩니다.

### 프로덕션 빌드 / 실행

```bash
npm run build      # shared → client → server 순서로 빌드
npm start          # 서버가 빌드된 클라이언트까지 서빙 (http://localhost:3001)
```

### 테스트

```bash
npm test           # 공유 엔진 단위 테스트 (vitest)
```

## 게임 규칙 (현재 구현 범위)

- **역할** (4~7인): 보안관 · 부관 · 무법자 · 배신자, 인원수별 자동 분배
- **승리 조건**: 보안관/부관(무법자·배신자 전멸) · 무법자(보안관 처치) · 배신자(최후 생존)
- **턴 진행**: 2장 뽑기 → 카드 플레이 → 손패 한도(=체력)까지 버리기
- **카드 (베이스 덱 전체)**
  - 공격/방어: 뱅!, 빗나감!, 나무통, 결투, 개틀링, 인디언의 습격
  - 회복/수급: 맥주, 술집, 역마차, 웰스파고, 잡화점
  - 견제: 약탈, 캣 발루, 감옥, 다이너마이트
  - 무기/장비: 볼캐닉·스코필드·레밍턴·카빈총·윈체스터, 머스탱, 조준경
- **거리/사거리**: 좌석 원형 거리 + 조준경(-1)/머스탱(+1), 무기별 사거리
